> Generator to angular with gulp

## Installation

First, install [Yeoman](http://yeoman.io) and generator-angular-ajax using [npm](https://www.npmjs.com/) (we assume you have pre-installed [node.js](https://nodejs.org/)).

```bash
npm install -g yo
npm install -g generator-angular-ajax
```

Then generate your new project:

```bash
yo angular-ajax
```
